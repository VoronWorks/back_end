Screenshots



  1. Подключение по SSH к учебному серверу
    При открытии программы Putty происходит SSH-подключение к удаленному серверу kubsu-dev.ru
    Происходит запрос логина и пароля.

  2. Работа команды ping
    При помощи этой команды узнаем IP-адрес веб-сервера kubsu.ru : 185.13.84.92 Команда определяет связь между 2 узлами: сервером kubsu.ru и подключенным сервером kubsu-dev.ru с помощью отправки небольших пакетов данных на kubsu.ru и проверяет задержку подключения к серверу

  3. Работа команды nslookup
    При помощи этой команды узнаем A-записи и MX-записи доменов kubsu.ru, kubsu-dev.ru
    Происходит запрос в домееной системе имен (DNS) с целью получения доменного инмени, IP-адреса или другой информации.
    A- записи содержат все возможные данные для указанного сервера.
    MX-записи содержат информацию о почтовых серверах(соответствие доменного имени почтовому серверу этого домена).

  4. Работа команды whois
    Даты регистрации: kubsu.ru 1998-03-28, kubsu-dev.ru 2020-02-12
    Команда показывает дату регистрации домена, информацию владельца домена, расположение и дату истечения срока домена.

  5. Клонирование репозитария на github на сервер в каталог /var/www/html/u52952 при помощи команды git clone
    Создается каталог c полной копией всех файлов и историей всех коммитов. В каталоге .git хранится сам репозитарий, остальное – файлы рабочей копии проекта.

  6. Соединение по протоколу FTP с использованием FileZilla и скачивание файлов на локальный компьютер
